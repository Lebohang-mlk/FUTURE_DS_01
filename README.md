# Online Retail Sales Analysis Dashboard

## Project Overview
This project presents a sales analysis of an online retail dataset using Microsoft Excel. The objective was to explore sales trends, identify top-performing countries and products, and build a visual dashboard to communicate insights effectively.

The project demonstrates skills in data cleaning, pivot table analysis, and dashboard development.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Dashboard Design

## Key Metrics
- Total Revenue: $9 747 747,93
- Total Orders: 541,910
- Average Order Value: $1.88

## Key Insights
- Sales increase toward the end of the year, with November showing the highest sales.
- The United Kingdom contributes the majority of total revenue.
- A significant portion of revenue is categorized under "Unspecified", indicating missing product descriptions in the dataset.
- Product revenue outside the "Unspecified" category is distributed across several items.

## Dashboard
This dashboard provides a visual overview of:
- Sales trends over time
- Revenue by country
- Top products by revenue
- Key performance indicators (KPIs)

## Project Purpose
The goal of this project is to demonstrate the ability to transform raw data into meaningful insights using Excel and present findings through a clear and interactive dashboard.
